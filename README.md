# SI140A.01-Project3 微信红包

## 项目概述

本仓库用于存储上海科技大学SI140A.01概率论课程的项目代码、文档和报告，主题为微信红包。

## 文件夹结构

```
├── code/             # 项目代码
├── docs/             # 项目文档
├── reference/        # 参考资料
├── report/           # 报告相关内容
│   ├── data/         # 报告数据
│   ├── image/        # 报告图片
│   ├── tex/          # LaTeX文件
│   └── upload_files/ # 上传文件
└── README.md         # 项目说明
```

## 各文件夹用途

### code/
- 存放项目的主要代码实现
- 包括数据分析、模拟实验、算法实现等
- 建议按功能模块或实验名称组织代码

### docs/
- 存放项目相关文档
- 包括需求文档、设计文档、用户手册等
- 支持Markdown、PDF、Word等格式

### reference/
- 存放项目参考资料
- 包括论文、书籍、博客等
- 建议按主题或来源组织

### report/
- 存放最终报告相关内容

#### report/data/
- 存放报告中使用的数据文件
- 包括原始数据、处理后的数据、结果数据等
- 支持CSV、Excel、JSON等格式

#### report/image/
- 存放报告中的图片文件
- 包括图表、示意图、截图等
- 建议使用PNG或PDF格式

#### report/tex/
- 存放LaTeX相关文件
- 包括报告的LaTeX源码、样式文件等

#### report/upload_files/
- 存放需要上传的文件
- 包括最终报告、演示文稿等

## 使用说明

1. 在code/文件夹中编写项目代码
2. 将生成的数据保存到report/data/文件夹
3. 将生成的图片保存到report/image/文件夹
4. 在report/文件夹中编写报告
5. 最终成果保存到report/upload_files/文件夹
